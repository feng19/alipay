# Alipay

**Alipay SDK for Elixir**

## Installation

You can be installed by adding `alipay` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:alipay, "~> 0.1", hex: :alipay_sdk}
  ]
end
```

## 快速测试

支付宝支持沙箱，用于快速测试

- [沙箱应用](https://open.alipay.com/develop/sandbox/app)
